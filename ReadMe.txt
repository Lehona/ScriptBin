ReadMe for ScriptBin

Um ScriptBin zu installieren, reicht es, die ScriptBin.src in der Gothic.src einzutragen (Ordner nicht vergessen!). Üblicherweise würde man das unter den Eintrag "_INTERN\CLASSES.D" einfügen.
Wer kein Ikarus benutzen möchte, öffnet die ScriptBin.src und löscht alles bis auf die erste Zeile (Zeilenumbruch am Ende nicht vergessen - sonst wird die Datei nicht geladen).
Wer auch LeGo-Scripte benutzen möchte, fügt die LeGo_Scripts\LeGo.src nach der Header.src aus LeGo in seine Gothic.src ein.

To install ScriptBin, simply add the ScriptBin.src to your Gothic.src. The best spot for this entry is right below "_INTERN\CLASSES.D".
If you don't use Ikarus, simply delete every line besides the first one in your ScriptBin.src. Don't forget the newline, otherwise it will not read the ScriptBin.src correctly.
If you want to use the LeGo scripts as well, add LeGo_Scripts\LeGo.src to your Gothic.src, right below Header.src from LeGo.
